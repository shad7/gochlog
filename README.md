# gochlog

Generate Change Log using Go

## Features

  * Generates Change Log output using `git log` metadata


## Usages

```bash
gochlog --help
```

#### Increase logging output

Levels: debug, info, warn, warning, error, fatal, panic

```
export GOCHLOG_LOG_LEVEL=<level>
```
